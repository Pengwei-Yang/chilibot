Chilibot Features

Overall
  • Extract term-term relationship from Medline abstracts.
  • Differentiate interactive (e.g. stimulation or inhibition) and non-interactive (e.g. homology, co-existence, etc.) interactions.
  • Color-code gene expression values when data are provided.

    Query

  • Accept gene symbol, gene name, UniGene ID, GenBank accession (slow, not recommended) and free form keywords as query term.
  • Automatic synonym lookup before PubMed query. Synonym database contains 53,542 entries.
  • Synonym is user editable.

    Visualization

  • Visualize interactions using 2 graph layout algorithms.
  • Generate subnetwork maps of various kinds.

    Information Summarization

  • Provide summary sentences for each term (synopsis), with user defined keywords hight-lighted.
  • Provide sentences describing the interaction of a pair of terms, with the terms and their synonyms highlighted.

    Other

  • Extensive links connect to PubMed for terms, synonyms, abstracts, relationships.
  • Links to other databases include OMIM, GeneCard, LocusLink, and Genome Database.
  • easy to use interface.
